Formatação

Tipos de formatação

Negrito

Para deixarmos o texto em negrito, podemos utilizar dois tipo de tags. A primeira, que é escrita da seguinte forma "b", é utilizada apenas para deixar o texto em negrito. E a segunda, que é escrita da seguinte forma "strong", que além de deixar o texto em negrito, é utilizada quando queremos dar ênfase em determinado trecho do texto.

Espaço

Quando queremos deixar um espaço entre texto usamos a código com & "nbsp" e ponto e virgula. Assim   como demostrado.

Italico

Quando queremos deixar o texto inteiro ou parte dele em italico, utilizamos a tag "i", e o texto fica com esta aparencia.

Sublinhado

Agora, se quiser deixar seu texto sublinhado, basta usar a tag "u".

Pular uma linha

Apesar de não ser muito inficado,

podemos fazer uma quebra de linha no HTML utilizando a tag "br". Mas está é apenas uma opção para ser utlizada no código HTML, o ideal é que a quebra de linha seja feita no CSS.

Texto pequeno

Agora quando queremos deixar um texto menor, usamos a tag "small". Essa opção é muito usado quando colocamos o titulo de uma obra e queremos que o autor apareça menor em baixo. Como no exemplo abaixo:

Sapiens - Uma breve história da humanindade

Yuval N. Harari

Texto riscado

Usamos essa tag "del" quando queremos corrigir algo que foi escrito certo errado. Muito utilizada em sites que fazems aqueles anuncios de x valor por y valor, como no exemplo abaixo:

De R$ 497,00 por R$ 300,00

Marcação

Utilizamos a tag "mark" de marcação, quando qeuremos que o texto, ou parte dele fique marcado. Essa função lembra muito um marca texto.

Linha de divisão

Quando queremos demarcar uma linha para dividir a pagina, ou um texto usamos a tag "hr".

Superior

A tag "sup" é usada para definir um texto sobrescrito, normalmente usado em notas de rosapé ou fórmulas matemáticas. Como no exemplo abaixo:

a 2 + b 2 = c 2

Inferior

A tag "sub" é usada para definir um texto subscrito, normalmente usado em fórmulas químicas. Como no exemplo abaixo:

A fómula química da água é H2O

Importante

Lembrando que com esxecão das tags "br" e "hr", todas as outras possuem abetura e fechamento. A abetura dela se dá pelo simbolo de menor, a tag e o simbolo de maior. E o fechamento ocorre quando colocamos o simbolo de menor, barra e o simbolo de maior.
