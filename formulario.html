Formulário

A tag "form" - Define o formulário;

A tag "label" - Define a legenda, texto que antecede a caixa do foumulário;

A tag "input" - Define a caixa de texto do formulário;

A tag "selec" - Define uma caixa de multipla escolha;

A tag "option" - Define as opções que irão aparecer no select;

A tag "textarea" - Uma caixa de texto maior, para digitar textos mais longos;

A tag "button" - Define a criação de um botão;

Atributo "id" - Define o idenficador;

Atributo "for" - Relaciona o atributo for ao campo de input através do ID;

Atributo "type" - Define o tipo do input(caixa de texto do formlário);

Atributo "placeholder" - Define o texto marca d'agua dentro da cx de resposta;

Atributo "name" - Define ;

Atributo "value" - Define a informação que será enviada junto com a resposta do usuário;

Atributo "cols" - Define a quantidade de colunas do textarea;

Atributo "rows" - Define a quantidade de linhas do textarea;

Atributo "required" - Torna o preenchimento do campo obrigatório;

Temos vários tipos de input, abaixo irei listar os principais:

  • Valor do atributo "type" - text - Digitar um texto, como nome ou sobrenome;
  • Valor do atributo "type" - email - Digitar o e-mail;
  • Valor do atributo "type" - password - Digitar a senha;
  • Valor do atributo "type" - radio - Criar uma lista de escolha única;
  • Valor do atributo "type" - checkbox - Criar uma lista de multipla escolha;
  • Valor do atributo "type" - submit - Botão de enviar formulário;
