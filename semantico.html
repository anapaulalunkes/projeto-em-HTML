Div e HTML Semântico

Quando falamos em tag div, estamos falando em divisórias, ou seja, como a minha página será dividida. Ex.: tem um topo para colocar um banner, o menu, e duas cxs de conteúdo. Mas o que dará vida ao layout da página é o CSS.

header é a tag semantica para o topo da página, usamos ela quando quando outra pessoa bater o olho no código, saiba que esse é o topo. A tag nav é um trecho do meu site usado em menu. section é a tag semantica para seção, uma parte da minha página. footer usada para fazer o rodapé. A tag article para dizer que é um artigo.

Podemos usar a tag div para fazer as divisórias, mas o nosso código fica de mais fácil compreensão quando usamos a forma semanatica correta.
